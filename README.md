## Utilisation

### Pré-requis

Python 2.7 et la librairie standard

### Jouer une partie

1. Rendre  `scrabble.py` exécutable :

    `chmod +x scrabble.py`

2. Exécuter  `scrabble.py`:

    `./scrabble.py`

3. Jouer en suivant les règles et le manuel d'utilisation (dans `doc/règles.md`)
